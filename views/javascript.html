Javascript

Targeting Elements

this - The DOM element the listener was attached to.
To get the desired functionality you must wrap it in a jQuery object like: $(this)
$(event.target) - The actual DOM element that was clicked
.event.preventDefault() - The default action of the event will not be triggered

Traversing the DOM

.find() - Selects the first element that matches the selector, down the DOM tree.
.closest() - Selects the first element that matches the selector, up from the DOM tree.
.parent() - Selects one element up the DOM tree.
.parents() - Method is similar to parent() but selects all the matching elements up the DOM tree.
.children() - Selects all the matching elements down the DOM tree.
.siblings() - Find element on same level

Inserting / Changing / Removing DOM Elements

.text() - Replaces the contents of the DOM element as plain text

Doing: 
    <div id="mydiv">Hello World</div>

    $('#mydiv').text('Aloha World');
Result: 
    <div id="mydiv">Aloha World</div>

.html() - Replaces the contents of the DOM element and the ability to add HTML characters

Doing: 
    <div id="mydiv">Hello World</div>

    $('#mydiv').html('Aloha World');
Result: 
    <div id="mydiv">Aloha World</div>

.replaceWith() - Removes content from the DOM and inserts new content in its place

Doing: 
    <div id="mydiv">Hello World</div>
  
    $('#mydiv').replaceWith('Aloha World');
Result: 
    Aloha World

.replace() - Searches a string for a specified value and replaces it with value provided 

    var str = "Visit Microsoft!";
    var res = str.replace("Microsoft", "W3Schools");

.filter() - Use filter() to select a specified set of elements 

    <ul>
    <li></li>
    <li class="middle"></li>
    <li></li>
    </ul>
    $("li").filter(".middle").css( "border-color", "red" );
.prepend() - Insert content to the beginning of the main element
.append() - Insert content to the end of the main element

Returning Data From Elements

.data() - The data() method attaches data to, or gets data from, selected elements

Doing: 
    <a href="#" id="mydiv" data-example="Goodbye">Hello World</a>
    <div id="result"></div>

    var newGreeting = $('#mydiv').data('example');
    $('#result').text(newGreeting);
Result: 
    <div id="result">Goodbye</div>

Animating Elements

.animate() - Create animation effects on any numeric CSS property

Doing: 
    $('#mydiv').animate({height: "20px"}, 500);
.slideDown() / .slideUp() - Display the matched elements with a sliding motion
.fadeIn() / .fadeOut() - Display the matched elements by fading them to opaque
